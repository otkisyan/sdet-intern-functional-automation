# IntelliJ IDE Starter - Changelists Settings UI Test

This project contains a Kotlin integration UI test based on JetBrains Starter/Driver.

Scenario covered:
1. Open an IntelliJ-based IDE.
2. Open a public test project from GitHub.
3. Open **Settings...**.
4. Go to **Version Control -> Changelists**.
5. Enable **Create changelists automatically**.
6. Verify that the checkbox is selected.
7. Click **OK**.
